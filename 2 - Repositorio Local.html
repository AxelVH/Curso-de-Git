Creación de un repositorio


        
            //Este comando sirve para inicializar un repositorio local
            git init

Creación de un repositorio

        
            //Sirve para ver en que estados estan los archivos del repositorio.
            git status
            //Sirve para agregar el archivo al estado "staging area".
            git add MiArchivo.html
            //Sirve para agregar todos los archivos al estado "staging area".
            git add .
            //Sirve para agregar todos los archivos del staging area al estado "repository".
            git commit -m "Mi primer commit"
            //Sirve para eliminar un archivo del estado "staging area".
            git restore --staged MiArchivo.html

Deshaciendo los cambios

        
            //Sirve para deshacer los cambios de un archivo y volver como estaba en el utlimo commit.
            git checkout MiArchivo.html
            //Sirve para deshacer los cambios de todo el repositorio y volver como estaba en el utlimo commit.
            git checkout .

Historial de cambios

        
            git log
            git log –oneline –all –graph --decorate

Regresando a una versión especifica

        
            git checkout HASH_DEL_LOG
