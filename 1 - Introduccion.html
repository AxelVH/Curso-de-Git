Configuración

El siguiente comando nos va a ayudar a asignarle a nuestro git algunas configuraciones a nivel global, esto quiere decir que esta configuración se va a mostrar para todos los usuarios de la computadora y en todos los proyectos. 


        
        git config --global user.email "axelvasquezherrera@outlook.com"
        git config --global user.name "Axel Yoliston Vasquez Herrera"

El siguiente comando nos ayuda a verificar la configuración asignada. 


        
        git config --global -l
        git config -list

Los estados de GIT

1 - Working Directory

Este estado es el espacio de trabajo

2 - Staging Area

Este estado es el area de preparación

3 - .git directory (Repository)

Es el estado del repositorio (carpeta .git)
